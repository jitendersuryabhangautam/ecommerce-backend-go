# E-Commerce Backend API

A full-featured e-commerce backend built with Go, PostgreSQL, and modern architectural patterns.

## Features

- ✅ User authentication & authorization (JWT)
- ✅ Product catalog with CRUD operations
- ✅ Shopping cart with stock reservation
- ✅ Order processing with state management
- ✅ Payment processing (mock implementation)
- ✅ Returns & refunds management
- ✅ Admin dashboard endpoints
- ✅ Rate limiting & request validation
- ✅ Comprehensive error handling
- ✅ Database migrations
- ✅ Docker support

## Tech Stack

- **Language**: Go 1.21+
- **Framework**: Gorilla Mux
- **Database**: PostgreSQL 15+
- **ORM**: pgx (PostgreSQL driver)
- **Authentication**: JWT
- **Validation**: go-playground/validator
- **Logging**: Built-in + custom middleware
- **Containerization**: Docker & Docker Compose

## Quick Start

### Prerequisites

- Go 1.21+
- PostgreSQL 15+
- Docker & Docker Compose (optional)

### Local Development

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   ```
